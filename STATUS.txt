==========================================

   - Fast pathogenic k-mer detection
   - O(1) lookup performance

2. DNABERT-2 Integration
   - 117M parameter transformer
   - Attention weight extraction

3. Hybrid Classifier
   - Combines Bloom + DNABERT features
   - Neural network classifier
   - Probability + confidence scores

4. Interactive Visualizations
   - Attention heatmaps
   - Nucleotide importance plots
   - Bloom filter hit overlay

5. Sickle Cell Focus
   - HBB gene variant detection
   - E6V mutation (sickle cell)
   - E6K mutation (HbC disease)

==========================================
EXAMPLE USAGE
==========================================

From Python:
```python
from bloom_dnabert import MultiScaleBloomFilter, DNABERTWrapper
from bloom_dnabert.classifier import HybridClassifierPipeline

bloom = MultiScaleBloomFilter()
bloom.load_hbb_pathogenic_variants()

dnabert = DNABERTWrapper()
pipeline = HybridClassifierPipeline(bloom, dnabert)

# Analyze a sequence
result = pipeline.predict("CACGTGGTCTACCCCTGAGGAG")
print(result['prediction'])  # Pathogenic or Benign
```

From Web Dashboard:

2. Navigate to http://localhost:7860
